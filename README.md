
# Password Manager 🔐

Локальный менеджер паролей с графическим интерфейсом. Проект построен на принципах чистой архитектуры, использует криптографию для защиты данных и хранит базу данных локально.

## ✨ Возможности

*   **Безопасное хранение:** Пароли шифруются алгоритмом AES-128-CBC (Fernet) на основе мастер-пароля.
*   **Организация данных:** Поддержка категорий для группировки паролей.
*   **История паролей:** Приложение запоминает старые пароли при их изменении.
*   **Удобный поиск:** Быстрый поиск по логину и названию сайта.
*   **Буфер обмена:** Копирование логинов и паролей в один клик.
*   **Статистика:** Встроенная панель статистики по сохраненным данным.

## 🛠 Технологии

*   **Язык:** Python 3.10+
*   **GUI Фреймворк:** PySide6 (Qt)
*   **ORM и База данных:** SQLAlchemy 2.0 + SQLite (`db.sqlite`)
*   **Криптография:** 
    *   `cryptography` (Fernet, PBKDF2HMAC) для симметричного шифрования данных.
    *   `bcrypt` для хеширования и проверки мастер-пароля.

## 🛡 Безопасность

*   **Мастер-пароль** никогда не хранится в открытом виде. В базу данных сохраняется только его `bcrypt` хеш с уникальной солью.
*   Ключ для шифрования данных генерируется "на лету" в оперативной памяти с помощью **PBKDF2HMAC**
*   Все сохраненные пароли в базе зашифрованы алгоритмом **Fernet**. Без мастер-пароля злоумышленник, получивший файл `db.sqlite`, не сможет прочитать ваши пароли.

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/yourusername/password-manager.git
cd password-manager
```

### 2. Создание виртуального окружения (рекомендуется)
```bash
python -m venv venv

# Активация на Windows:
venv\Scripts\activate
# Активация на macOS/Linux:
source venv/bin/activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Запуск приложения

При первом запуске база данных `db.sqlite` создастся автоматически.

**Для запуска введите в консоль из корня проекта:**
```bash
python main.py
```

## 📝 Использование

1.  **Регистрация:** При первом входе выберите опцию «Регистрация» и придумайте надежный мастер-пароль. Запомните его! Без него вы потеряете доступ к своим данным.
2.  **Добавление:** Нажмите «Добавить пароль», введите сайт, логин и пароль. Вы можете назначить одну или несколько категорий через запятую.
3.  **Просмотр:** В главном окне используйте кнопку 👁 (глаз), чтобы показать пароль, или ⋮ (меню) для быстрого копирования в буфер обмена.
